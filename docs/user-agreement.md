## 一、定义与关于我们

### 1. 产品定义

翠虎远程桌面是一款用于管理远程连接并访问远程桌面会话的客户端工具。本应用主要提供以下功能：

- 创建、保存、编辑、删除和搜索远程连接配置；
- 通过 VNC 协议连接远程桌面；
- 支持可选的 SSH 隧道能力，包括密码认证、公钥认证、主机校验与端口转发；
- 支持远程桌面画面查看、鼠标/触控/键盘控制；
- 支持文本剪贴板同步；
- 支持多屏识别、屏幕切换、显示布局管理与查看模式切换；
- 支持截图保存、本地状态提示、安全等级提示和必要的开发诊断辅助功能。
